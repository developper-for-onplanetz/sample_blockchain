## 2-6: Import

### 概要
#### Importとは

- コードが長くなりすぎないよういくつかのファイルを作った時に別のファイルからインポートする時に利用する
- import "./ファイル名";

---

### コード
#### 問題  

1. 新しく作ったzombiefeeding.solからzombiefactory.sol をインポートせよ。

``` :zombiefeeding.sol
// import ステートメントをここに書け
```

#### 正解

``` :zombieFeeding.sol
// import ステートメントをここに書け
import "./zombiefactory.sol";
```
